# Instrucciones para configurar el frontend

1. Crea un archivo llamado '.env.local' en esta carpeta con el siguiente contenido:

```
NEXT_PUBLIC_API_URL=http://localhost:3000/api
NEXT_PUBLIC_AUTH0_DOMAIN=tu-dominio-auth0.auth0.com
NEXT_PUBLIC_AUTH0_CLIENT_ID=tu-client-id-auth0
NEXT_PUBLIC_AUTH0_REDIRECT_URI=http://localhost:3000/api/auth/callback
```
